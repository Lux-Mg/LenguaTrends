# Руководство пользователя — LenguaTrends

Краткое руководство по работе с веб-интерфейсом программной системы анализа трендов комментариев на различных языках.

## Запуск системы

См. `README.md` в корне репозитория. После запуска интерфейс доступен по адресу `http://localhost:3000`.

## Навигация

В верхней части экрана расположена панель навигации с семью вкладками. Справа от вкладок находится переключатель языка интерфейса (RU / ES / EN). Выбранный язык влияет на:

- Подписи элементов интерфейса.
- Названия кинопроизведений (отображаются в локализованной форме, если доступна).
- Текст в карточках, заголовках и подсказках.

Язык интерфейса не зависит от языка анализируемых комментариев — пользователь может выбрать русский интерфейс и при этом просматривать данные на испанском.

## Описание вкладок

### Главная

Сводный обзор корпуса:

- **KPI-панели** в верхней части: общее число комментариев, число фильмов, число обработанных комментариев, наиболее обсуждаемое кинопроизведение.
- **Топ-5 трендов** — таблица с пятью самыми обсуждаемыми фильмами и их сентимент-показателями.
- **Распределение по языкам** — процентное соотношение комментариев на каждом из трёх языков.
- **Последние комментарии** — лента свежих записей с указанием тональности.

### Тренды

Полный рейтинг кинопроизведений с фильтрами:

- **Язык** — фильтрация по языку комментариев (Все / EN / ES / RU).
- **Период** — последние 7, 30 или 90 дней.
- **Фильм** — выбор конкретного фильма из выпадающего списка.

В таблице отображается ранг, название, число упоминаний, доля позитивных комментариев и общая тональность.

### Тональность

Анализ распределения тональности на двух уровнях:

1. **По языкам** — гистограмма показывает распределение позитивных, негативных и нейтральных комментариев в каждой языковой аудитории.
2. **По фильмам** — топ-5 кинопроизведений по числу комментариев с цветовой индикацией тональности.

Дополнительно отображаются три карточки: самый позитивный язык, самый позитивный фильм, наиболее критикуемый фильм.

### Межъязыковые различия

Сопоставительный анализ восприятия фильмов разными языковыми аудиториями.

В верхней части находится **переключатель минимального числа комментариев на язык** (50 / 75 / 100 / 150). Чем выше значение, тем более статистически надёжны результаты, но тем меньше фильмов попадает в выборку.

Раздел содержит шесть информационных карточек:

- **Относительный фаворит русских / испанцев / англичан** — фильм, который данная языковая аудитория оценила выше других на максимальный процент позитивных комментариев.
- **Самый спорный фильм** — фильм с максимальной разницей в оценке между языками.
- **Наибольшее согласие между языками** — фильм с минимальным разбросом оценок (все три языка пришли к единому мнению).
- **Самый ценимый в среднем** — фильм с максимальной средней долей позитивных комментариев среди всех языков.

Ниже расположена таблица с детальным разбором: для каждого фильма показаны процент позитивных комментариев и размер выборки на каждом языке, общий разброс, а также языки-«победитель» и «проигравший».

### Облако слов

Частотный анализ лексики.

- **Селектор языка** позволяет переключаться между испанским, английским и русским словарями.
- Размер слова в облаке пропорционален частоте употребления.
- Цвет слова отражает доминирующую тональность контекста, в котором это слово встречается.

Применяются стоп-слова трёх категорий: общеязыковые, специфическая лексика обзоров YouTube и нормализация диакритических знаков (для испанского — обработка вариантов с диакритикой и без).

### Динамика

Временной ряд активности по неделям.

Линейный график показывает количество комментариев в неделю в разрезе трёх языков. Помогает увидеть пики обсуждений (например, неделя релиза фильма).

Боковые карточки сообщают пик активности, наиболее активный язык и среднюю активность за неделю.

### Комментарии

Просмотр сырых данных с возможностью фильтрации.

Доступные фильтры:
- Язык комментария.
- Период.
- Фильм.
- Тональность (позитив / негатив / нейтрал).

В таблице отображаются: тональность, язык, текст комментария, фильм, оценка модели. Реализована пагинация.

## Типичные сценарии использования

### Сценарий 1. Сравнение восприятия одного фильма разными аудиториями

1. Перейти на вкладку **Тренды**.
2. Выбрать интересующий фильм в фильтре «Фильм».
3. Перейти на вкладку **Тональность**, она уже фильтрована по выбранному фильму.
4. Изучить распределение тональности по языкам.
5. Перейти на вкладку **Межъязыковые различия**, найти фильм в таблице — увидеть точные цифры разрыва между языками.

### Сценарий 2. Поиск культурно-специфических предпочтений

1. Перейти на вкладку **Межъязыковые различия**.
2. Выставить минимальное число комментариев на язык — 75 или 100 для надёжности.
3. Изучить три карточки «Относительный фаворит» — каждая показывает, какой фильм нашёл наибольший относительный отклик именно у данной языковой аудитории.

### Сценарий 3. Детальное изучение пользовательских мнений

1. Перейти на вкладку **Облако слов**, выбрать интересующий язык.
2. Найти выделяющиеся часто употребляемые слова.
3. Перейти на вкладку **Комментарии**, отфильтровать по нужному языку и тональности.
4. Прочитать живые комментарии для понимания контекста.

## Известные ограничения

- Корпус на 23.04.2026 содержит 19 155 комментариев по 16 кинопроизведениям. Для редких или нишевых фильмов выборка на отдельных языках может быть недостаточной (документированный пример — фильм Mike & Nick & Nick & Alice, для которого русскоязычный сегмент составляет всего 10 комментариев).
- Анализ тональности использует мультиязычную модель XLM-RoBERTa, оптимизированную для коротких текстов социальных сетей. Точность может снижаться на длинных и формальных текстах.
- Система не выполняет анализ комментариев на языках, отличных от ES, EN и RU. Такие комментарии помечаются меткой `unsupported` и исключаются из обработки.
